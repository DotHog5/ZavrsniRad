Pravljenje elemenata

Pravljenje odnosno kreiranje elementa se ogleda u tome da pomoću Javaskripta napravimo html tag sa klasama, atributima i sadržajem i postavimo na stranicu.


const p = document.createElement('p')

console.log(p)

-> <p></p>

/*Za početak ćemo deklarisati promenljivu, kojoj ćemo dodeliti vrednost createElement()*/


Zatim dodajemo klasu i id na element


p.className = 'collection-item'

p.id = 'new-item'

console.log(p)

-> <p class="collection-item" id="new-item"></p>


Potom dodajemo atribute


p.setAttribute('title', 'New Item')

console.log(p)

-> <p class="collection-item" id="new-item" title="New Item"></p>


Moramo dodati i text node i da appendamo

Text node je zapravo tekst koji se nalazi između 2 HTML taga(element node je zapravo sam HTML element). Append znači da dodamo taj element


p.append(document.createTextNode('Pozdrav!'))

console.log(p)

-> <p class="collection-item" id="new-item" title="New Item">Pozdrav!</p>


Sad ćemo kreirati još jedan element i dodati njega na prethodni i sve to ispisati na stranicu


const link = document.createElement('a')

link.setAttribute('href', 'http://www.google.com')

link.innerHTML = ' Klikni ovde'

p.appendChild(link)
