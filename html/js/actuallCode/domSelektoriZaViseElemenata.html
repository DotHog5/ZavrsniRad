DOM selektori za više elemenata

Kao što se može videti iz naslova u ovoj lekciji ćemo raditi višestruke selektore elemenata


getElementByClassName()


Svim elementi koji imaju istu klasu imamo pristup pomoću getElementByClassName().


<ul id="list">

<li class="list-element list-element">1. Primer</li>

<li class="list-item" id="list">2. Primer</li>

<li class="list-element">3. Primer</li>

<li class="list-item list-element">4. Primer</li>

<li class="list-item">5. Primer</li>

<li class="list-element">6. Primer</li>

<li class="list-element">7. Primer</li>

<li class="list-item">8. Primer</li>

</ul>

<script>

console.log(document.getElementsByClassName('list-element'))

</script>

-> HTMLCollection(5) [li.list-element.list-element, li.list-element, li.list-item.list-element, li.list-element, li.list-element]

/*Dobijamo HTML kolekciju, broj članova u kolekciji i koji su to članovi.*/


Kod ovog selektora moramo dodatno korisitit petlju, jer je dobijeni rezultat kolekcija.


getElementByTagName()


Svim elementi koji imaju isti tag imamo pristup pomoću getElementByTagName().


<ul id="list">

<li class="list-element list-element">1. Primer</li>

<li class="list-item" id="list">2. Primer</li>

<li class="list-element">3. Primer</li>

<li class="list-item list-element">4. Primer</li>

<li class="list-item">5. Primer</li>

<li class="list-element">6. Primer</li>

<li class="list-element">7. Primer</li>

<li class="list-item">8. Primer</li>

</ul>

<script>

const liElement = document.querySelectorAll('.list-element')

const liItem = document.querySelectorAll('.list-item')

liElement.forEach(function (li) {

li.style.background = 'grey'

})

liItem.forEach(function (li) {

li.style.background = 'skyblue'

})

</script>

  • 2. Primer

querySelectorAll()


Za razliku od querySelector(), querySelectorAll() selektuje sve elemente sa datom klasom ili tagom. Omogućava nam jednostavan pristup i menjanje grupe HTML elemenata sa istom klasom ili tagom.


<ul id="list">

<li class="list-element list-element">1. Primer</li>

<li class="list-item" id="list">2. Primer</li>

<li class="list-element">3. Primer</li>

<li class="list-item list-element">4. Primer</li>

<li class="list-item">5. Primer</li>

<li class="list-element">6. Primer</li>

<li class="list-element">7. Primer</li>

<li class="list-item">8. Primer</li>

</ul>

<script>

let prom = document.querySelectorAll('.list-element')

prom.forEach(function(item, index){

item.textContent = `${index}: Zdravo`

})

</script>

0: Zdravo

2. Primer

1: Zdravo

2: Zdravo

5. Primer

3: Zdravo

4: Zdravo

8. Primer

/*Brojevi izgledaju ovako razbacano jer su samo elementi sa klasom list-element promenjeni i ispred njih stoji broj njihovog indexa*/
