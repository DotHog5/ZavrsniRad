If Iskaz

Kod if iskaza izvrsiće se jedan deo koda, ako je ispunjen uslov, ili neki drugi iskaz, ako je kod njega ispunjen uslov.


const id = 100;

if (id == 100){

console.log("Tačno!")

} else {

console.log("Netačno!")

}

-> Tačno

/*Moramo obratiti pažnju da NE stavimo jedan znak jednakosti, jer on predstavlja dodelu vrednosti, a nama treba znak za poredjenje!!*/


Takođe možemo proveriti da li neka vrednost nije jednaka:

if (id != 100){

console.log("Tačno!")

} else {

console.log("Netačno!")

}

-> Tačno!


Poredjenje po vrednosti i tipu

Ako želimo da poredimo i po vrednosti i po tipu onda stavljamo tri znaka jednakosti(===).


if (id === "100"){

console.log("Tačno!")

} else {

console.log("Netačno!")

}

-> Netačno!

/*Stavljanjem navodnika smo pretvorili 100 u string i time dobijamo kao rezultat netačno, jer je id tipa int*/


Provera da li neki iskaz nije jednak po vrednosti i tipu:


if (id !== "100"){

console.log("Tačno!")

} else {

console.log("Netačno!")

}

-> Tačno!


Veće ili manje

if (id > 150){

console.log("Tačno!")

} else {

console.log("Netačno!")

}

-> Netačno!


if (id < 150){

console.log("Tačno!")

} else {

console.log("Netačno!")

}

-> Tačno!


Else if

Pored if i else imamo i treću mogućnost, else if. Else if možemo da koristimo koliko puta želimo, u zavisnoti koliko iskaza nam treba.


const boja = "plava"

if (boja === "crvena"){

console.log("Boja je crvena!")

} else if(boja === "zelena") {

console.log("Boja je zelena!")

} else if(boja === "bela") {

console.log("Boja je bela!")

} else{

console.log("Boja nije ni crvena, ni zelena, ni bela.")

}

-> Boja nije ni crvena, ni zelena, ni bela.


Logički operateri


const ime = "Ivan"

const god = 19


And &&

if (god > 0 && god < 12){

console.log(`${ime} je dete`)

/*Ovde uvodimo template literal, koji nam omogućava da koristimo neku promenljivu, a da pritom ne pišemo njenu vrednost direktno u kod. Takođe ovde se primenjuju ``, a ne ""!*/

} else if(god >= 13 && god <= 19) {

console.log(`${ime} je tinejdžer`)

} else{

console.log(`${ime} je odrastao`)

}

-> Ivan je tinejdžer


Or ||

if (god < 16 || god > 65){

console.log(`${ime} ne može da se trka`)

} else{

console.log(`${ime} može da se trka`)

}

-> Ivan može da se trka


Terinarni operater

Predstavlja isto if iskaz, ali u skraćenom obliku

console.log(id === 100 ? "Tačno" : "Netačno")

-> Tačno
