Petlje

Petlje nam omogućavaju da jedan deo koda ponavljamo, sve dok se uslov ne ispuni.


For petlje

for(let i = 0; i < 10; i++){

console.log("Broj: " + i)

}

-> Broj: 0

Broj: 1

Broj: 2

Broj: 3

Broj: 4

Broj: 5

Broj: 6

Broj: 7

Broj: 8

Broj: 9


Prva i najkorišćenija petlja jeste for petlja. Sastoji se iz tri dela: initializer(let i = 0), test condition (i < 10) i updater (i++), on obično povećava ili smanjuje vrednost. For petlja se koristi kada znamo tačan broj iteracija, koliko puta treba da se ponovi petlja.


for(let i = 0; i < 10; i++){

if (i === 4){

console.log("4 je moj omiljeni broj")

}

}

-> 4 je moj omiljeni broj

While petlja

Kod while petlje prvo se postavi uslov, ako je on tačan uslov se ponavlja sve dok ne bude netačan. Ovde obavezno moramo obratiti pažnju na updater, jer ako ga ne deklarišemo dobićemo beskonačnu petlju!


let i = 0

while(i < 10){

console.log("Broj " + i)

i++

}

-> Broj: 0

Broj: 1

Broj: 2

Broj: 3

Broj: 4

Broj: 5

Broj: 6

Broj: 7

Broj: 8

Broj: 9

Do while petlja

Razlika između while i do while jeste da se do while uvek barem jednom izvrši.


let x = 0

do {

console.log("Broj " + x)

x++

}

while(x < 10)

-> Broj: 0

Broj: 1

Broj: 2

Broj: 3

Broj: 4

Broj: 5

Broj: 6

Broj: 7

Broj: 8

Broj: 9

For in petlja


For in petlja ide kroz vrednosti objekta i ispisuje ih


let student = {

ime: "Nemanja",

god: 21,

smer: "IT"

}

for(let x in student){

console.log(`${x} : ${student[x]}`)

}

-> ime : Nemanja

god : 21

smer : IT
