Konkatenacija

Konkatenacija predstavlja spajanje stringova. Pored toga ćemo obraditi još neke funkcionalnosti stringova, u ovoj lekciji.

Radi jednostavnosti koristićemo uvek iste vrednosti:


const ime = 'Jovana'

const prezime = 'Đorđević'

const godine = 19

const str = 'Zdravo moje ime je Nemanja'

let rezultat


Konkatenacija

rezultat = ime + prezime

console.log(rezultat)

-> JovanaĐorđević

/* bolje rešenje: */

rezultat = ime + " " + prezime

console.log(rezultat)

-> Jovana Đorđević

rezultat = "Zdravo, moje ime je " + ime + " i imam " + godina + " godina"


Append

rezultat = 'Jovana '

rezultat += 'Đorđević'

console.log(rezultat)

-> Jovana Đorđević

/* Dodavanje novog stringa na već postojeći. */


Length

rezultat = ime.length

console.log(rezultat)

-> 6

/* Ispisuje se koliko karaktera ima string */


Concat

rezultat = ime.concat(" ", prezime)

console.log(rezultat)

-> Jovana Đorđević

/* Prvi parametar je šta će se nalaziti između 2 stringa. */


Promena veličine slova

rezultat = ime.toUpperCase()

console.log(rezultat)

-> JOVANA

rezultat = ime.toLowerCase()

console.log(rezultat)

-> jovana


Ispisivanje slova

rezultat = ime[0]

console.log(rezultat)

-> j

/* Ispisuje prvo slovo(slovo sa indeksom 0) */


indexOf()

rezultat = ime.indexOf('n')

console.log(rezultat)

-> 4

/* Ispisuje index karaktera. */

rezultat = ime.lastIndexOf('n')

console.log(rezultat)

-> 1

/* Ispisuje index karaktera od nazad. */


substring()

rezultat = ime.substring(0, 4)

console.log(rezultat)

-> Jova

/* Ispisuje karaktere sa indeksima datim u zagradi, do vrednosti drugog parametra. */


slice()

rezultat = ime.slice(0, 4)

console.log(rezultat)

-> Jova

/* Slično substringu, koristi se najviše kod nizova. */


split()

rezultat = str.split(" ")

console.log(rezultat)

-> Array(5) [ "Zdravo", "moje", "ime", "je", "Nemanja"]

/* Pravi niz od stringa, gledajući reči koje su odvojene, u ovom slučaju razmakom, ali može bilo šta da bude parametar */


replace()

rezultat = str.replace("Nemanja", "Milan")

console.log(rezultat)

-> Zdravo moje ime je Milan


includes()

rezultat = str.includes("Zdravo")

console.log(rezultat)

-> true

/* Vraća vrednost true ili false u zavisnoti da li je ta reč sadržana ili ne. */
