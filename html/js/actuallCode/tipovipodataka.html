Tipovi Podataka


Promenljiva u Javaskriptu može da sadrži bilo koji podatak. U jednom trenutku može biti string, u drugom number:


let message = 'Zdravo'

message = 12345

console.log(message)

-> 12345


Programski jezici koji omogućavaju ovo se nazivaju dinamički. Tipove Podataka u Javasktiptu možemo podeliti na primitivne i referentne.


Primitivni Tipovi Podataka

Primitivni tipovi(skalarni ili jednostavni tipovi) čuvaju podatke( npr. string -> "primer"; number -> 123) i njihova vrednost se direktno čuva u memoriji. Postoje 6 vrsta primitivnih tipova podataka:

  • - string
  • - number
  • - boolean
  • - null
  • - undefined
  • - symbol

String

String je tip podataka koji u sebi sadrži niz slova/reči, takođe može i brojeva, ali se onda neke funkcionalnosti gube.


const ime = 'Marko Ivanić'

console.log(typeof ime)

-> string

/* Sa typeof u konzoli dobijamo naziv tipa podataka */


Number

Number je tip podataka koji u sebi čuva brojeve.


const broj = 33

console.log(typeof broj)

-> number


Boolean

Tip podataka koji može biti true ili false


const bool = false

console.log(typeof bool)

-> false


Null

Null ne znači da je vrednost promenljive 0, već da nema vrednost


const auto = null

console.log(typeof auto)

/* Rezultat će biti object, ali to je bug, jer se null direktno čuva u memoriji i čuva jednostavne podatke*/


Undefined

Ukoliko ne dodelimo vrednost dobijamo undefined


let test

console.log(typeof test)

-> undefined


Symbol

Novi tip podataka, koji je sa ES6 uveden. Symboli su uvek unikatni, njihove vrednosti ne mogu biti iste, npr:


let Sym1 = Symbol("Sym")

let Sym2 = Symbol("Sym")

console.log(Sym1 == Sym2)

-> false


let Sym1 = Symbol("Sym")

console.log(typeof Sym1)

-> symbol


Referentni Tipovi

Zbog toga što se sadržaj referentnog tipa ne može uklopiti u fiksnu količinu memorije koja je dostupna za promenljivu, vrednost memorije referentnog tipa je sama referenca (memorijska adresa).

Postoje 3 vrste referentnih tipova podataka:

  • - niz
  • - objekat
  • - date

Niz

Koristi se da bi se u jednoj varijabli čuvalo više vrednosti.


const hobi = ['film', 'muzika']

console.log(typeof hobi)

-> object


Objekat

Objekat može sadržati više vrednosti. Piše se u vidu ime : vrednost para.


const adresa = {

grad: "Novi Sad",

država: "Srbija"

};

console.log(typeof adresa);

-> object


Date

Stvara instancu koja predstavlja trenutak u formatu nezavisnom od platforme. Objekti koji sadrže datum, sadrže broj koji predstavlja milisekunde od 1. januara 1970. UTC


const danas = new Date()

console.log(typeof danas)

-> object
