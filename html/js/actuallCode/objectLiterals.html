Object Literals

Object Literal u Javaskriptu predstavlja listu naziv-vrednost parova, koji su odvojeni zarezima i nalaze se u vitičastim zagradama. Smanjuje upotrebu globalnih promenljivih koji bi mogli izazvati smetnju.


Evo i primera:


const osoba = {

ime: "Ivan",

prezime: "Marković",

godine: 21,

email: "marko@gmail.com",

hobiji: ["sport", "čitanje"],

adresa: {

država: "Srbija",

grad: "Novi Sad"

},

getGodinuRodjenja: function() {

return 2019 - this.age;

} //dobijamo godinu rođenja osobe

};


Kao što možemo videti u object literal možemo da stavi bilo koje podatke, čak i druge objekte i funkcije.


Sve ove vrednosti možemo ispisati:


console.log(osoba.godine)

-> 21

console.log(osoba.ime)

-> Marko

console.log(osoba.hobiji[1])

-> čitanje

console.log(osoba.adresa.država)

-> Srbija

/*Sve vrednosti se mogu ovako ispisivati*/
