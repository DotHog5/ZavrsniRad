Keyboard Event

Keyboard eventovi, kao što se može pretpostaviti, odigravaju se nakon pritiska tastera na tastaturi. To znači da ih koristimo prilikom unosa teksta u input polja.


Keydown


<form>

<div class="form-group">

<input class="form-control w-25" type="text" id="ime" placeholder="Upišite nešto">

</div>

<p class="h3" id="provera">Test</p>

</form>

<script>

let btn = document.getElementById('ime')

btn.addEventListener('keydown', kydwn)

let provera = document.getElementById("provera")

function kydwn(){

provera.style.color = "red";

}

</script>
