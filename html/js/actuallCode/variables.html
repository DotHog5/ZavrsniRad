Deklarisanje Promenljivih


Postoje tri vrste deklaracije promenljivih: var, let i const. Var je već poprilično zastareo, umesto njega se koriste let i const. let je sličan var-u, uz razliku što se let može jednom samo proglasiti i var se koristi za za deklarisanje u globalnom smislu, dok je let limitiran u području bloka , npr:


var name = "Marko Ivanić";
console.log(name);
var name = "Jovana Đorđević";
console.log(name);

-> Marko Ivanić
Jovana Đorđević


let name = "Marko Ivanić";
console.log(name);
let name = "Jovana Đorđević";
console.log(name);

-> SyntaxError
/* let name je već definisan i ne može se nova vrednost dodeliti istoj promenljivoj.*/


Ovo takođe ne može:


let name;
let name = "Jovana Đorđević";
console.log(name);

-> SyntaxError
/* Isto, kao i u prošlom primeru, let je već definisan, iako mu nije dodeljena vrednost(prima vrednost undefined).*/


const


const se koristi da se istoj promenljivoj ne bi dodelili 2 puta različite vrednosti:


const name = "Marko Ivanić";
name = "Jovana Đorđević";

-> TypeError

/*Nije moguće dodeliti različite vrednosti jednom const-u, u const stavljamo vrednosti koje znamo da se neće menjati kroz kod.*/


Takođe const moramo deklarisati, inače ćemo dobiti error:


const name;
console.log(name);

-> SyntaxError
