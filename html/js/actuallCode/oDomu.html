O DOM-u

Šta je zapravo DOM

Document Object Model(DOM) definiše standard za pristup dokumentu. Omogućava dinamični pristup i izvršavanje promena sadržaja, struktura i stilova dokumenta.

Bilo koja veb stranica jeste dokument. Taj dokument se može prikazati ili putem veb pretraživača ili HTML koda, ali je isti dokument u oba slučaja. DOM predstavlja taj isti dokument, ali na način da se s njim manipuliše.

Jedan od osnovnih primera jeste sa getElementById:


let param = document.getElementById("id")

/*param će dobiti pristup HTML elementu sa id-om "id"*/


DOM je ugrađeni API i bez njega Javaskript ne bi imao toliku funckionalnost i ne bi bio toliko popularan. Svaki element dokumenta - dokument u celini, glava, telo, tabele - sve je to deo Document Object Modela za taj dokument. Svim tim delovima se može pristupiti i manipulisati pomoću DOM-a i skriptnog jezika kao što je Javaskript.

DOM je dizajniran da bude nezavisan od bilo kog drugog programskog jezika, omogućavajući pristup istom kroz jedan API.


Pristup DOM-u


Pristupi DOM-u je veoma jednostavno i ništa komplikovano nije potrebno za njega. Različiti veb pretraživači imaju različite implementacije DOM-a, ali u suštini svaki pretraživač koristi neki document object model dopuštajući pristup stranici putem Javaskripta.


Napomena: Budući da se velika većina koda koji koristi DOM vrti oko manipulisanja HTML dokumenta, uobičajeno je da nodove u DOM-u uvek nazivamo elementima, jer je u HTML dokumentu svaki node element.
