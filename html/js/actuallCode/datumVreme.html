Datum i Vreme

Ispisivanje trenutnog datuma


const danas = new Date()

console.log(danas)

-> Tue Aug 13 2019 17:07:43 GMT+0200 (Central European Summer Time)


Postavljanje datuma


const datumRodjenja = new Date("11/6/1997")

console.log(datumRodjenja)

-> Thu Nov 06 1997 00:00:00 GMT+0100 (Central European Standard Time)

datumRodjenja = new Date("September 11 2005")

-> Sun Sep 11 2005 00:00:00 GMT+0200 (Central European Summer Time)

Uzimanje vrednosti datuma

Mesec


console.log(danas.getMonth())

-> 7

/*Januar je 0, decembar je 11*/


Dan


console.log(danas.getDay())

-> 2

/*Nedelja je 0, ponedeljak 1...*/


Godina


console.log(danas.getFullYear())

-> 2019


Sati


console.log(danas.getHours())

-> 17


Minuti


console.log(danas.getMinutes())

-> 19


Sekunde


console.log(danas.getSeconds())

-> 23


Milisekunde


console.log(danas.getMilliseconds())

-> 524


Postavljanje vrednosti datuma

Mesec


datumRodjenja.setMonth(2)

console.log(datumRodjenja)

-> Fri Mar 11 2005 00:00:00 GMT+0100 (Central European Standard Time)


Dan


datumRodjenja.setDate(12)

console.log(datumRodjenja)

-> Fri Mar 12 2005 00:00:00 GMT+0100 (Central European Standard Time)


Godina


datumRodjenja.setFullYear(1966)

console.log(datumRodjenja)

-> Fri Mar 12 1966 00:00:00 GMT+0100 (Central European Standard Time)


Sati


datumRodjenja.setHours(3)

console.log(datumRodjenja)

-> Fri Mar 12 1966 03:00:00 GMT+0100 (Central European Standard Time)


Minuti


datumRodjenja.setMinutes(33)

console.log(datumRodjenja)

-> Fri Mar 12 1966 03:33:00 GMT+0100 (Central European Standard Time)


Sekunde


datumRodjenja.setSeconds(51)

console.log(datumRodjenja)

-> Fri Mar 12 1966 03:33:51 GMT+0100 (Central European Standard Time)
