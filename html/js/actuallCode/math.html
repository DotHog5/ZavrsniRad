Javaskript Math Objekat

Omogućava korišćenje matematičkih radnji u kodu.


Proste matematičke radnje

const num1 = 100

const num2 = 50

let val

val = num1 + num2

-> 150

val = num1 / num2

-> 2

val = num1 - num2

-> 50

val = num1 * num2

-> 5000

val = num1 % num2

-> 0


Math Objekat


PI

val = Math.PI

-> 3.141592653589793

/* Vrednost broja PI */


E

val = Math.E

-> 2.718281828459045

/* Vrednost broja e */


Zaokruživanje

val = Math.round(2.8)

-> 3


Zaokruživanje uvek na veći broj

val = Math.ceil(2.4)

-> 3


Zaokruživanje uvek na manji broj

val = Math.floor(2.8)

-> 2


Korenovanje

val = Math.sqrt(64)

-> 8


Apsolutna vrednost

val = Math.abs(-17)

-> 17


Stepenovanje

val = Math.pow(8, 2)

-> 64

/* Prvi broj je onaj koji se stepenuje,a drugi predstavlja stepen */


Minimalna vrednost

val = Math.min(2, 33, 42, 11, 24)

-> 2


Maksimalna vrednost

val = Math.max(2, 33, 42, 11, 24)

-> 42


Random

val = Math.random()

-> Nasumično će ispisati neku decimalu, koja je uvek manja od 1


Random od 1 do 20

val = Math.floor(Math.random() * 20 + 1);

/* Ovako se generiše nasumični broj od 1 do 20(uključujući i 20) */
