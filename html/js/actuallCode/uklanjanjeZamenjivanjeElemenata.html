Uklanjanje i zamenjivanje elemenata

Zamenjivanje elemenata


Za početak ćemo prvo napraviti novi element i apendovati ga.


<div class="parent-class">

<h2 class="text-title" id="title">Pozdrav</h2>

</div>

<script>

const noviNaslov = document.createElement('h2')

noviNaslov.id = 'title'

noviNaslov.appendChild(document.createTextNode('Naslov'))

</script>


Zatim pomoću selektora pristupamo elementu koji hoćemo da promenimo i njegovom parentu.


const stariNaslov = document.getElementById('title')

const parent = document.querySelector('.parent-class')


parent.replaceChild(noviNaslov, stariNaslov)

Uklanjanje elemenata


Kod uklanjanja moramo prvo selektovati željene elemente pomoću selektora i brišemo ih tako što zapravo uklanjamo elemente kolekcije.


<ul>

<li>1. Primer</li>

<li>2. Primer</li>

<li>3. Primer</li>

<li>4. Primer</li>

<li>5. Primer</li>

<li>6. Primer</li>

<li>7. Primer</li>

<li>8. Primer</li>

</ul>

<script>

const li = document.querySelectorAll('li')

const list = document.querySelector('ul')

</script>


Potom sledi i uklanjanje elemenata


li[0].remove()

/*Ukloniće element sa indexom 0, odnosno prvi element*/

list.removeChild(li[3])

/*Uklanja child element sa indexom 3, odnosno četvrti element*/
