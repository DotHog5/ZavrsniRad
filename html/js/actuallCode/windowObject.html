Window object


Alert


alert("Pozdrav")

-> Iskočiće prozor sa porukom: "Pozdrav"


Prompt


const input = prompt()

alert(input)

-> Iskočiće prozor u koji treba da upišemo neki tekst, a zatim će taj isti tekst biti ispisan u novom prozoru.


Confirm


if (confirm('Da li si siguran?')) {

console.log('da');

} else {

console.log('ne');

}

-> Ispisaće se da ili ne, u zavisnosti da li kliknemo ok ili cancel


Svojstva

Ovde ćemo wideti koja sva svojstva ima window object.


Spoljašnja visina i širina


console.log(window.outerHeight)

/*Ispisuje spoljašnju visinu stranice*/


console.log(window.outerWidth)

/*Ispisuje spoljašnju širinu stranice*/


Unutrašnja visina i širina


console.log(window.innerHeight)

/*Ispisuje unutrašnju visinu stranice*/


console.log(window.innerWidth)

/*Ispisuje unutrašnju širinu stranice*/


Location Object


console.log(window.location)

/*Ovom komandom u konzoli nam se ispusuje razni podaci o dokumentu, kao što su hostname, putanja dokumenta, protokoli...*/


Možemo i zasebne vrednosti da uzimamo


console.log(window.location.port)

console.log(window.location.hostname)

console.log(window.location.protocol)


Redirect


console.log(window.location.href = 'http://google.com')

/*Prebacuje nas na drugu stranicu*/


console.log(window.location.reload())

/*Konstantno refrešuje stranicu*/


History object


Ovo je read-only property, koji vraća referencu na History object, koji nam pruža interfejs za manipulisanje sesije web browser istorije.


console.log(window.history.back())

/*Jednako dugmetu unazad*/

console.log(window.history.go(-1))

/*Jednako history.back*/
