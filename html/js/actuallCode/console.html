Ispisivanje u konzolu


Komandom console.log() ispisujemo željeni sadržaj u console-u. Koristi za debugging, proveru da li nešto radi kako treba.


console.log('Hello World!')

-> Hello World


Ovu komandu smo već prešli, takođe pored teksta možemo i ispisivati brojeve:

console.log(33)

-> 33


Postoji i mogućnost da stavimo brojeve u navodnike, ali onda bi se učitali kao string i neke druge funckionalnosti nam ne bi bile dostupne:

console.log('33')

-> 33

/* Iako izgleda isto sa i bez navodnika, razlika je velika, o tome u nekom od sledećih poglavlja. */


Ispisivati možemo i booleane, true ili false

console.log(true)

-> true

console.log(false)

-> false


U console.log() možemo smestiti i promenljivu:

var pozdrav = "Zdravo!"

console.log(pozdrav)

-> Zdravo!


Nizove takođe možemo ispisivati, treba naglasiti da u Javaskriptu niz ne postoji kao tip podataka, to je ustvari tip object

console.log([1, 2, 3, 4])

-> [1, 2, 3, 4]


zbog toga što je niz ustvari objekat, možemo napisati i sledeće

console.log({ a: 1, b: 2 });

-> {a: 1, b: 2}

Ovo pretstavlja object literal, kao što je napomenuto to je objekat


Pomoću console možemo ispisivati i erore

console.error("Ovo je eror");

-> Ovo je eror

/*Eror će biti ispisian crvenim slovima u crvenom pravougaoniku.*/


Pored erora imamo i warninge

cconsole.warn("Ovo je upozorenje");

-> Ovo je upozorenje

/*Upozorenje će biti ispisano crnim slovima u žutom pravougaoniku.*/
