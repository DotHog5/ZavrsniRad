Nizovi

Nizovi čuvaju podatke u redosledu: prvi, drugi, treći... U niz se može smestiti bilo šta, brojevi, karateri, imena, objekti i sl. Svaki element niza ima svoj indeks, prvi element ima indeks 0, drugi 1 itd.


Deklaracija

Postoje dva načina da se deklariše niz:


let arr = new Array()

let arr = []


Skoro pa uvek se upotrebljuje drugi način.

Kao što smo rekli u niz može bilo šta da se postavi.


const brojevi = [33, 1, 11, 25, 44, 23, 17]

/*Brojevi*/

const string = ["Pera", "Jabuka", "Zgrada", "Teren"]

/*Stringovi*/

const numbers = ["Zdravo", 7, true, undefined, null, { a: 1, b: 2 }, new Date()]

/*Mešano*/


*U sledećim primerima koristićemo niz brojevi kao primer.


Dužina niza

console.log(brojevi)

-> 7


Provera da li je niz

console.log(Array.isArray(brojevi))

-> true

/*Vraća vrednost true ili false u zavisnosti da li je u pitanju niz ili ne.*/


Uzimanje jedne vrednosti

console.log(brojevi[3])

-> 11


Zamena vrednosti

console.log(brojevi[2] = 100)

-> 100


Pronalaženje indexa niza

console.log(numbers.indexOf(33));

-> 0

/*Ovde naravno moramo znati da li ta vrednost postoji u nizu, ako ne vratiće vrednost -1*/


Menjanje strukture nizova

Dodavanje na kraj niza

brojevi.push(150)

console.log(brojevi[7])

-> 150

/*Zamenjuje poslednji član niza sa novom vrednošću.*/


Dodavanje na početak niza

brojevi.unshift(12)

console.log(brojevi[1])

-> 12

/*Zamenjuje prvi član niza sa novom vrednošću.*/


Uklanjanje sa kraja

brojevi.pop()

console.log(brojevi)

-> [33, 1, 11, 25, 44, 23]

/*Uklanja poslednju vrednost u nizu*/


Uklanjanje sa početka

brojevi.shift()

console.log(brojevi)

-> [1, 11, 25, 44, 23, 17]

/*Uklanja prvu vrednost u nizu*/


Uklanjanje od-do

brojevi.splice(1,3)

console.log(brojevi)

-> [33, 44, 23, 17]

/*Uklanja vrednosti u nizu od indexa 1 do indexa 3, računajuči i vrednost sa indexom 3*/


Postavljanje niza od nazad ka napred

brojevi.reverse()

console.log(brojevi)

-> [17, 23, 44, 25, 11, 1]


Konkatenacija nizova

console.log(brojevi.concat(string))

-> [17, 23, 44, 25, 11, 1, "Pera", "Jabuka", "Zgrada", "Teren"]

/*Dodaje sve članove niza na kraj prvog.*/


Sortiranje nizova

console.log(brojevi.sort())

console.log(string.sort())

-> [1, 11, 17, 23, 25, 33, 44]

/*Ređanje od najvećeg ka najmanjem*/

-> ["Jabuka", "Pera", "Teren", "Zgrada"]

/*Ređanje po abecedi*/
