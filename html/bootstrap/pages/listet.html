Liste

Liste se koriste da bi se izlitao bilo koji sadržaj u njima. U prvom primeru će biti prikazana najednostavniji tip liste.


<ul class="list-group">

<li class="list-group-item">Primer Liste Jedan</li>

<li class="list-group-item">Primer Liste Dva</li>

<li class="list-group-item">Primer Liste Tri</li>

<li class="list-group-item">Primer Liste Četiri</li>

<li class="list-group-item">Primer Liste Pet</li>

</ul>

Kontekstualne liste

Slična funkcija običnim listama, uz razliku što su tekst i pozadina u drugačijim bojama

<li class="list-group-item">Regular List Item</li>

<li class="list-group-item list-group-item-primary">Primary List Item</li>

<li class="list-group-item list-group-item-secondary">Secondary List Item</li>

<li class="list-group-item list-group-item-success">Success List Item</li>

<li class="list-group-item list-group-item-info">Info List Item</li>

<li class="list-group-item list-group-item-warning">Warning List Item</li>

<li class="list-group-item list-group-item-danger">Danger List Item</li>

<li class="list-group-item list-group-item-light">Light List Item</li>

<li class="list-group-item list-group-item-dark">Dark List Item</li>

</ul>

Badges

Sa bedževima možemo da dodamo npr. aktivnosti, nepročitane poruke i sl.

<li class="list-group-item">Primer Liste Jedan</li>

<li class="list-group-item d-flex justify-content-between align-items-center">Primer Liste Dva <span class="badge badge-info">33</span></li>

<li class="list-group-item">Primer Liste Tri</li>
