Dobro došli na stranicu priručnika za Bootstrap


Šta je zapravo Bootstrap

Bootstrap predstavlja besplatni web framework otvorenog koda, za kreiranje veb sajtova i veb aplikacija. Baziran je na HTML i CSS šablonima za tipografiju, kreiranju formulara, dugmadi, navigacionim i ostalim komponentama interfejsa, kao i opcionim Javaskript dodacima. Cilj Bootstrap framwrork-a je olakšavanje programiranja za veb. Takođe, Bootstrap je framework za veb aplikacije, tj. software framework koji je dizajniran tako da podrži razvoj dinamičkih veb stranica i veb aplikacija.


Kako koristiti Bootstrap

Bootsrap imai svoj veb sajt, https://getbootstrap.com/ Tu se mogu naći i biblioteke koda, ukoliko skinemo Bootstrap, takođe ga možemo jednostavno pridodati našem kodu, tako što ubacimo Bootstrap CDN, odnosno linkove, a zatim koristimo klase koje nam Bootsrap i pruža.
